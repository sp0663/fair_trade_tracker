Backward Lineage Tracing — Coffee Retail to Origin

Traces Coffee Retail 1 (batch 106) backward through every supply-chain stage to its raw-material origins at the farm level. The recursive CTE walks the batch_relations DAG until no more parents are found; UNION (not UNION ALL) collapses the same node visited through multiple paths.
Recursive Backward Lineage Query (batch_id = 106)
WITH RECURSIVE lineage AS ( SELECT b.id, b.product_name, s.name AS owner, s.role, 0 AS depth FROM batches b JOIN stakeholders s ON b.current_owner_id = s.id WHERE b.id = 106 UNION SELECT parent.id, parent.product_name, ps.name, ps.role, l.depth + 1 FROM batches parent JOIN stakeholders ps ON parent.current_owner_id = ps.id JOIN batch_relations br ON br.parent_batch_id = parent.id JOIN lineage l ON l.id = br.child_batch_id ) SELECT * FROM lineage ORDER BY depth DESC, id;
